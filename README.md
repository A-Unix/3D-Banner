## Installation instructions

 ### Clone the repo
  ```bash
  git clone https://github.com/Atuls-git/3D-Banner.git
  ```
 ### Move to cloned folder
  ```bash
  cd 3D-Banner
  ```
 ### Get permissions
  ```bash
  chmod +x *
  ```
 ### Install dependencies
  ```bash
  ./setup.sh
  ```
 ### Run the main file
  ```bash
  python3 main.py
  ```
Enjoy :)
